# V2

**Active design** toward a Mustache-inspired, [logic-less presentation](./logic-less-presentation.md) template language with a JavaScript/TypeScript-first [host layer](./host-layer.md), optimized for **code generation** use cases.

V2 is **not** a synonym for [modern syntax](./modern-syntax.md): the experimental `parseModern` grammar is exploratory. Normative goals are in [V2 design goals](../features/architecture/v2_design_goals.md); the on-paper language specification is in [V2 language spec](../features/language-spec/index.md). Decision record: [ADR-002](../features/architecture/adr-002-mustache-js-first-code-generation.md); [Pkl](https://pkl-lang.org/) is **not** a recommended path.

[Legacy syntax](./legacy-syntax.md) remains the stable shipped surface until V2 syntax lands.
